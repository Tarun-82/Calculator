# 🧮 Calculator Web App

A simple and responsive Calculator built using HTML, CSS, and JavaScript. This project performs basic arithmetic operations and provides a clean user interface for quick calculations.

## 🚀 Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Decimal Number Support
- Clear Display Function
- User-Friendly Interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Structure

```
Calculator/
│
├── index.html
├── index.css
└── README.md
```

## 📸 Preview

Simple calculator interface with numeric buttons, operators, display screen, and calculation functionality.

## 🎯 How to Run

1. Clone the repository

```bash
git clone <repository-url>
```

2. Open the project folder

```bash
cd Calculator
```

3. Open `index.html` in your browser
